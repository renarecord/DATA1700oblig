Kort beskrivelse av applikasjon:
Dette prosjektet er en enkel webapplikasjon for registrering og håndtering av kinobilletter. Brukere kan lagre, hente, oppdatere og slette billetter via et REST API.
Applikasjonen er utviklet med Spring Boot, og bruker Spring Data JPA for databasehåndtering med en in-memory H2-database.
